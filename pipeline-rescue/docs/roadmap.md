# Roadmap

## Phase 0 - Foundation

- lock positioning
- lock MVP scope
- define pricing
- define core scoring rules
- define HubSpot data contract
- define pilot event model

## Phase 0.5 - Executable specification

- finalize scoring thresholds by pipeline profile
- lock required scopes and object mappings
- define reason taxonomy
- define next-best-action mapping
- define pilot dashboards and baseline queries

## Phase 1 - Pilot build

- HubSpot OAuth app shell
- deal analysis endpoint
- deal card UI
- task creation
- draft generation
- internal admin logging

## Phase 2 - Pilot customers

- install on 3 to 5 real portals
- measure adoption
- refine score and copy
- capture objection patterns

## Phase 3 - Marketplace listing

- tighten scopes
- complete listing assets
- publish marketplace app
- collect install proof and reviews

## Phase 4 - Expansion

- manager digest
- pipeline rescue queue
- workflow automation refinements
- Salesforce port
- Microsoft Copilot port

## Success metrics

- weekly active rescued deals
- tasks created from recommendations
- drafts accepted by reps
- recovery rate on at-risk deals
- pilot retention

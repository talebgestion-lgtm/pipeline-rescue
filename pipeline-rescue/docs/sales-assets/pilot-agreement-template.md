## Pilot

Product: Pipeline Rescue

Pilot duration: `30` days from the activation date.

Pilot fee:

- setup: `EUR500`
- 30-day pilot: `EUR299`

## Scope

Provider will configure a controlled Pipeline Rescue pilot for the customer.

Included:

- HubSpot connection through OAuth or Private App token
- stalled deal analysis
- at-risk deal queue
- task creation workflow
- draft generation with human review
- HubSpot note write-back
- weekly review
- basic support by email

Excluded:

- autonomous message sending
- custom CRM migration
- custom HubSpot consulting beyond pilot setup
- production SLA
- guaranteed recovered revenue
- legal compliance guarantee

## Customer responsibilities

Customer must:

- provide access to a suitable HubSpot account or token
- confirm which pipeline is in scope
- avoid entering sensitive personal data in operator notes
- review all generated drafts before use
- provide feedback during the pilot

## Data handling

The pilot should use only business sales data required for stalled deal analysis.

The product should not be used for:

- health data
- legal advice data
- special-category personal data
- sensitive employee data

## Human review

Customer remains responsible for reviewing all recommendations, tasks, and drafts before acting on them.

Pipeline Rescue does not auto-send customer communications in the pilot.

## Success review

At the end of the pilot, both parties review:

- analyzed deals
- at-risk deals
- tasks created
- drafts generated
- useful/dismissed feedback
- observed renewed activity

## Termination

Either party may stop the pilot if:

- CRM data is not usable
- access cannot be maintained
- recommendations are consistently not credible
- compliance concerns cannot be resolved

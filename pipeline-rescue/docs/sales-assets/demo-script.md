# Demo Script

## Objective

Show that Pipeline Rescue can turn HubSpot stale-deal data into a clear human-reviewed action.

Keep the demo under `15` minutes.

## Before the call

- confirm HubSpot is the active sales CRM
- confirm the buyer has real open deals
- ask for one sample stale deal if appropriate
- avoid discussing broad SaaS promises

## Demo flow

### 1. Set the frame

"Pipeline Rescue is not an autonomous AI SDR. It is a focused workflow for stalled deals already inside HubSpot."

### 2. Show the live queue

Show:

- analyzed deals
- at-risk deals
- critical deals
- recovered revenue candidate
- ranked queue

Say:

"The first question is not what to send. It is which deals deserve attention today."

### 3. Open one deal

Show:

- deal name
- owner
- rescue score
- risk level
- top reasons
- normalization warnings

Say:

"If the system cannot trust a field, it reports that instead of inventing certainty."

### 4. Show recommended action

Show:

- recommended next action
- task body
- due date

Say:

"The goal is to reduce manager and rep ambiguity: what should happen next?"

### 5. Create HubSpot task

Click task creation.

Say:

"This writes back into HubSpot, so the rescue action becomes part of the existing CRM workflow."

### 6. Generate draft

Show:

- subject
- body
- human review requirement

Say:

"No auto-send in v1. The draft is there to save time, not remove judgment."

### 7. Save note

Save the draft as a HubSpot note.

Say:

"The CRM keeps the trace of what was recommended and drafted."

### 8. Show feedback and manager report

Show:

- useful/dismissed feedback
- reason themes
- manager digest

Say:

"The pilot is not just about generating actions. It is about learning which recommendations are trusted by the team."

## Close

"The pilot decision is simple. We run this on one pipeline for 30 days and measure whether at-risk deals receive faster, more consistent follow-up."

## Avoid saying

- "This will recover X revenue."
- "This replaces your managers."
- "The AI can run your sales process."
- "The system is fully autonomous."
- "Compliance is guaranteed."
